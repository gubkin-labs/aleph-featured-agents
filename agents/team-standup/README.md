# Team Standup Digest

**Who it’s for:** Teams that want an async standup without another SaaS standup bot.

**What it does on a schedule:**

- **09:00 UTC** — posts the shipped / blocked / next prompt.
- **16:00 UTC** — summarizes updates from memory into a digest.

**What to set after clone:**

1. Enable the agent (ideally under an org or team scope).
2. Tell it team members and norms in chat.
3. Connect Discord or Telegram under **Channels** so prompts and digests land in the team chat.
4. Have people reply with their updates during the day (chat or channel).

**Channel tip:** Connect one team Discord channel or Telegram group and fan-out schedules will deliver there when bindings are active.

No vault secrets required for the agent itself.
